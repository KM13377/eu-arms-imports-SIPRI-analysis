# EU Arms Imports & Dependency Analysis (SIPRI-based)

## Research Question
How has European dependence on external arms suppliers evolved after 2022?

## Methodology
This project uses SIPRI-based approximations of arms import dependency to analyse changes in defence procurement patterns across major European NATO members.

## Key Findings
- European NATO members increased reliance on US defence systems after 2022.
- Poland shows the strongest increase in external dependency.
- Western European states show moderate but consistent increases.
- The shift reflects accelerated NATO rearmament after Russia’s invasion of Ukraine.

## Data Source
- SIPRI Arms Transfers Database
